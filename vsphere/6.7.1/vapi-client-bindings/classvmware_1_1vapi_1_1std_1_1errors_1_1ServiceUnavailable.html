Detailed Description

The ServiceUnavailable error indicates that the interface is unavailable.

Examples:

  • Attempt to invoke a method when the server is too busy.
  • Attempt to invoke a method when the server is undergoing maintenance.
  • An method fails to contact VMware Tools running inside the virtual machine.

Counterexamples:

  • A client provides an invalid service or operation identifier when invoking the method using a dynamic interface (for example REST). The OperationNotFound error would be used instead.
  • A client invokes the method from the interface, but that interface has not been installed. The OperationNotFound error would be used instead.
